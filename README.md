# HotelBookingz
node/react hotel books


# set up
run `npm install`
run `cd web/` && `npm install`
run `cd ../`
run `docker-compose up -d --build mongo`

Find docker image ID {id} by running `docker ps`
run `docker -it {id} bash`
run `mongo -u root -p` and enter `example` as password
